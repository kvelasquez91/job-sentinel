# Contributing

This repository is a **generated mirror**: every commit on `main` is produced
by a gated publishing pipeline from a private working repository (the gate
strips the owner's personal data — that's why history starts where it does
and why commits carry a neutral author). A PR merged here directly would be
overwritten by the next publish and would bypass that gate, so:

- **Issues: welcome.** Bug reports, source breakage, ideas — file away.
- **PRs: reviewed, then landed by re-expression.** If accepted, the change is
  applied in the private repo and republished. The publish commit credits you
  with `Co-authored-by: <your-handle> <id+handle@users.noreply.github.com>`
  (GitHub-native attribution). If your handle collides with the gate's
  blocklist (rare), the commit credits the PR number instead ("lands PR #N").
- This is a personal project published reference-first: PRs are reviewed as
  time permits, without turnaround promises. Forking is encouraged — it's MIT.
